# Spots

An image sharing site.
  
**Description**
  
This design is made so all the elements are displayed effectively on popular screen sizes. Almost every element is responsive and moves based on the size of the browser or screen. 
  
  **Tech Stack**  
  
 * HTML
 * CSS
 * Responsive Design
